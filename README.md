# MachineLearning

I have taken "120 years of Olympic history: athletes and results" Data set from Kaggle and 
implemented 1.  EDA
            2.  KNN
            3.  K Means
            4.  Hierarchical Clustering
            5.  Decision tree
            6.  Logistic Regression
            7.  SVM
